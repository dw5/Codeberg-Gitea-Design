Welcome to the Codeberg Design example!

You can find all required information to use this design in the Codeberg Design Wiki.
This page is basically the sample page for Halfmoon CSS with the Codeberg stylesheet.

design.codeberg.org acts as a CDN for design resources, and contains the following embeddable resources:

  • /design-kit/codeberg-design.css
  • /design-kit/halfmoon.min.js
  • /design-kit/halfmoon.min.css (scoped version of Halfmoon CSS, automatically imported by codeberg-design.css)
  • /logo-kit/icon.svg (only the logo graphic without text)
  • /logo-kit/icon.png
  • /logo-kit/icon_inverted.svg (white on dark background)
  • /logo-kit/icon_inverted.png
  • /logo-kit/stacked.svg (logo with text below)
  • /logo-kit/stacked.png
  • /logo-kit/stacked_inverted.svg
  • /logo-kit/stacked_inverted.png
  • /logo-kit/horizontal.svg (logo with text besides)
  • /logo-kit/horizontal.png
  • /logo-kit/horizontal_inverted.svg
  • /logo-kit/horizontal_inverted.png

A CDN for fonts hosted on Codeberg can be found at fonts.codeberg.org.
